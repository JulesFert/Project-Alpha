The project

This is a 'wiki' type website that collects information on cryptids, living species whose existence is presumed, supported by various kinds of evidence, yet insufficient for recognition by the scientific community. It features creatures like the popular Loch Ness Monster, or the equally famous Yeti. It is structured around a homepage that redirects to articles, which are collaboratively created by the community. The site provides all the necessary tools for writing and editing articles by its members. It operates with a public section, accessible to everyone, and a private section, reserved for the administrative team, where more critical functionalities for the operation of the site can be used.
